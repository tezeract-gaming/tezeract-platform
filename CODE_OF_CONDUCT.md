# Code of Conduct

We use the [**Contributor Covenant**](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
version 2.1, verbatim, as the code of conduct for this project.

Short version: be respectful, assume good intent, and don't make this
project unwelcoming. Public spaces include GitHub issues, pull requests,
discussions, and any Tezeract-organized event.

## Reporting

If you experience or witness behaviour that breaks the Contributor
Covenant, report it confidentially to
**conduct@tezeract.com**.

All reports are reviewed by the project maintainers and treated
seriously. Reporters' identities are protected to the extent compatible
with investigating the report. We aim to respond within three business
days.

## Enforcement

Maintainers may remove comments, commits, code, wiki edits, issues, and
other contributions that don't follow the Contributor Covenant.
Sustained or severe violations result in a temporary or permanent ban
from project spaces.

## Scope

This code applies in all project spaces (this repo, related GitHub
discussions, conference booths at Tezeract-sponsored events) and when
an individual is representing the project or its community in public
spaces.
